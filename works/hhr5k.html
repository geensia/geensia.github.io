Project Overview

Hmong Heritage Run 5K (HHR5K) is a nonprofit program that promotes health and wellness in the Hmong community while also preserving the Hmong culture. I designed and developed a WordPress website that aligned with the HHR5K brand identity and provided users with easy access to event information, registration details, sponsorship opportunities, and program updates. Because the target audience most likely have access to the website on mobile devices, I prioritized a responsive design that delivered an easy and seamless experience across all types of devices.

Through this project, I learned to use a new platform, WordPress, to build and manage websites while also strengthening my understanding of responsive design, brand consistency, and user experience.
